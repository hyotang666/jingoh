# [Function] WRONG-FORMAT-P

## Syntax:

(WRONG-FORMAT-P arg) => result

## Arguments and Values:

arg := any lisp object

result := boolean

## Description:
Tests arg is wrong-format object or not.
When arg is wrong-format object, returns T, otherwise NIL.
