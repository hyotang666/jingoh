## Syntax:

(UNMATCH-CONDITION-P arg0) => result

## Arguments and Values:

arg := any lisp object

result := boolean

## Description:
Tests arg is unmatch-condition object or not.
When arg is unmatch-condition object, returns T, otherwise NIL.
