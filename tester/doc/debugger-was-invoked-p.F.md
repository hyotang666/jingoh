# [Function] DEBUGGER-WAS-INVOKED-P

## Syntax:

(DEBUGGER-WAS-INVOKED-P arg) => result

## Arguments and Values:

arg := any lisp object

result := boolean

## Description:
Tests arg is debugger-was-invoked object or not.
When arg is debugger-was-invoked object, returns T, otherwise NIL.
