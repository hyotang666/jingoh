# [Variable] `*READ-PRINT*`

## Value Type:

BOOLEAN

## Initial Value:

NIL

## Description:
Like `CL:*COMPILE-PRINT*` and `*CL:*LOAD-PRINT*`, `*READ-PRINT*` prints meaningful information to `CL:*TRACE-OUTPUT*`.
This is refered by `|#?reader|`.
