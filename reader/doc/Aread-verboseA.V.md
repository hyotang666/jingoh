# [Variable] `*READ-VERBOSE*`

## Value Type:

BOOLEAN

## Initial Value:

NIL

## Description:
Like `CL:*COMPILE-VERBOSE*` and `CL:*LOAD-VERBOSE*`, `*READ-VERBOSE*` prints meaningful information to `CL:*TRACE-OUTPUT*`.
This is refered by `|#?reader|`.
